# Shorten Url

This project provide a shorten url service. After running the service, you can open the url http://localhost:3000 from your prefered browser.





## Setting backend url

```
src/vars.ts
```

### Build frontend interface
```
npm run build
```

### Run Shorten Url Service
```
docker-compose up
```

### Run your unit tests
```
npm run test
```
